# Installation

[lazy.nvim](https://github.com/folke/lazy.nvim)

```lua
{
    "gmr458/awesomedark_theme.nvim",
    lazy = false,
    priority = 1000,
    config = function()
        require("awesomedark").setup({
            cursorline = true,
            transparent_background = false,
            nvim_tree_darker = true,
        })
        vim.cmd.colorscheme("awesomedark")
    end,
}
```
